The Calculator

Explanation of Calculator Functions

The C and CE buttons clears the calculator of anything you hit but once you hit equals it will save in the history list below the calculator.

The % percent button changes any number into a decimal form.

The <= back space button just takes away the current input but one space at a time.

The / division button divides two numbers.

The * multiplication button multiplies two numbers.

The - subtraction button subtracts two numbers.

The + addition button adds two numbers.

The 1/x fraction button takes any number you input and puts it into a fraction and gets the total for you.

The ^ exponent button allows you to add an exponent to a number.

The square root button takes the square root of any number.

The +/- positive and negative button allows you to change any number hit into a negative or positive number.

The = equal button calculates your function and saves it in the history list.

PLEASE HIT THE EQUALS SIGN AFTER EVERY TIME YOU DO A EQUATION TO SAVE IT TO THE HISTORY LIST!
